# 角色与道德议题（主题）

本文件详细说明约翰·特鲁比（John Truby）故事写作体系中的“角色网络设计”与“道德议题（主题）规划”。

---

## 一、角色网络 (Character Network)

角色不是孤立存在的，而是相互关联并定义彼此的网络。

### 1. 依功能构建角色网络
* **主角 (Protagonist)：** 追寻欲望并经历核心转变的角色。
* **对手 (Opponent)：** 与主角争夺相同目标，且最能直击主角弱点的角色。
* **盟友 (Ally)：** 协助主角达成目标的人。盟友不只是主角的传声筒，也应该有自己的欲望。
* **对手／假盟友 (Opponent/Fake-Ally)：** 表面是盟友，实则是对手或为主要对手服务（如《北非谍影》的雷诺局长）。这能增加剧情的复杂度与逆转空间。
* **盟友／假对手 (Ally/Fake-Opponent)：** 表面是对立面，实则是真心帮助主角的人。
* **副线角色 (Subplot Character)：** 用于跟主角的处境进行比较。

### 2. 角色原型 (Character Archetypes)
* **国王／父亲 (King/Father)：** 代表秩序、保护、统治，但也可能流于专制。
* **皇后／母亲 (Queen/Mother)：** 代表生育、培育、关怀，但也可能流于控制或窒息。
* **智慧老人／智慧老妇／导师 (Wise Old Man/Woman/Mentor)：** 提供知识与指引。
* **武士 (Warrior)：** 代表行动、保护、捍卫理想，但也可能好斗。
* **魔法师／巫师 (Magician/Wizard)：** 掌控看不见的力量，推动转变。
* **骗术师 (Trickster)：** 通过恶作剧或混乱揭露真相，打破僵局。
* **艺术家／小丑 (Artist/Clown)：** 创造或娱乐，代表自由心灵。
* **爱人 (Lover)：** 代表情感连结、美与奉献。
* **叛逆者 (Rebel)：** 挑战旧秩序，代表变革。

### 3. 主角的四个核心要求
* **始终保持吸引力：** 主角不一定是好人，但必须有某种迷人或强大的特质。
* **观众认同但适可而止：** 观众应能代入角色，但不能使其完美无瑕。
* **引发同理心而非同情：** 同理心（理解并感受）比同情（怜悯）更能拉近与观众的距离。
* **同时具备心理与道德需求：** 这是最重要的要求。主角必须有性格缺陷，且此缺陷伤害了他人（道德弱点）。

### 4. 四角对立 (Four-Corner Opposition)
这是特鲁比最核心的角色设计技巧：

```
       [主角 (Protagonist)] <===============> [对手 A (Opponent A)]
              ^                                      ^
              |                                      |
              |                                      |
              v                                      v
       [对手 B (Opponent B)] <===============> [对手 C (Opponent C)]
```

* **规则：**
  1. 将主角与至少三个对手放在同一个冲突网中。
  2. 确保每个角色的价值取向都有所不同。
  3. 让每个对手都与主角以及其他对手产生冲突。
  4. 避免简单的二元对立（善恶对决），而要展现不同道德层次和生存哲学的碰撞。

## 二、道德议题与主题 (Theme / Moral Argument)

主题是作者的道德愿景，即你对人应如何待人处世的看法。不要让角色成为你的传声筒，而应通过结构和冲突来传达。

### 1. 道德议题的基本策略
* **道德议题等于剧情：** 剧情的推进过程就是道德辩论的过程。主角的行动会产生道德后果，这些后果又会引发对手的反击与盟友的批判。
* **将角色置于价值冲突中：** 每个角色都代表一种生存价值观（如：自由 vs 安全，个人利益 vs 集体义务）。让这些价值观在情节中直接碰撞。

### 2. 道德议题的变奏与层次
* **善／恶 (Good vs. Evil)：** 最基础的层次，通常见于神话和童话。
* **悲剧 (Tragedy)：** 主角意识到自己的道德错误，但为时已晚，最终走向毁灭（如《麦克白》、《俄狄浦斯王》）。
* **动之以情 (Pathos)：** 聚焦于普通人在无法抗拒的环境下的道德挣扎。
* **讽刺／嘲讽 (Irony/Satire)：** 揭露角色的虚伪，展现行动的讽刺性后果（主角想达到 A，但其手段反而导致了相反的 B）。
* **黑色喜剧 (Black Comedy)：** 以幽默方式呈现荒谬而残酷的道德困境。

### 3. 双重逆转 (Double Reversal)
* 为了让道德议题更具说服力，赋予主角和主要对手双方的“真实自我揭露”。
* 对手不应是扁平的坏人，他必须有其人性、需求，且能从对决中有所体会。
* 主角和对手各自从对方身上学到东西，而两者学习的总和，就是作者最终要传达的道德愿景。
