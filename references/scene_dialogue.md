# 场景编排与交响乐式对白

本文件详细说明约翰·特鲁比（John Truby）故事写作体系中的“场景编排（Scene Weave）”与“场景构建与交响乐式对白设计”。

---

## 一、场景编排 (Scene Weaving)

场景编排是开始写作场景前的最后一步。它要求作者列出故事中的所有场景，并将主要剧情线（Plotline）、副线（Subplot）和主题线交织成一张漂亮的织锦。

### 1. 场景编排的四大原则
* **因果衔接：** 确保上一个场景的结果直接导致下一个场景的发生，避免事件无关地堆叠。
* **步调控制 (Pacing)：** 根据张力交替安排高压场景（如冲突、揭露）与低压场景（如思考、盟友对话）。
* **主题对照：** 在相邻场景中呈现不同角色面对相似道德困境的选择，强化主题深度。
* **空间流动：** 配合主角的心理状态，使场景在自然空间、人造空间或不同微缩世界之间转换。

### 2. 不同类型的场景编排模式
* **多线式/群戏编排：** 交替推进不同角色线，但在关键时刻让它们交汇（如《急诊室的故事》、《天人交战》）。
* **侦探犯罪故事编排：** 围绕“线索发现 -> 嫌疑人审问 -> 假象破除 -> 真相大白”的逻辑链条，逐步扩大揭露（如《洛城机密》）。
* **交叉剪接编排 (Cross-Cutting)：** 两个或多个行动在不同地点同时进行，借此将张力推向顶点（如《星球大战：帝国反击战》中卢克与达斯·维达的决斗与叛军的撤退交叉进行）。
* **爱情故事编排：** 围绕“相遇 -> 价值对立 -> 试探 -> 误会/假盟友干扰 -> 真实剖白”的轨迹，编排情感的起伏（如《傲慢与偏见》）。

## 二、场景构建 (Scene Construction)

写作场景时，每个场景本身也必须是一个小型的结构体。

### 1. 场景构建的结构步骤
* **场景目标 (Scene Objective)：** 确定角色在这个场景中想要得到什么。
* **直接计划 (Direct Plan)：** 角色为达到该场景目标采取的即时策略。
* **冲突 (Conflict)：** 对手或环境如何阻碍主角。
* **价值取向的碰撞 (Value Conflict)：** 场景中冲突的实质——双方信守的价值观对立。
* **结局 (Ending)：** 场景的结果，通常是主角面临新的障碍或获得小幅揭露。

### 2. 复杂与潜文本场景 (Subtext)
* 角色在场景中说的话（文本）和他们真正想表达的意图（潜文本）不一致。
* 透过眼神、动作、象征性物件或双关语来表现潜文本，让场景产生耐人寻味的张力。

## 三、交响乐式对白 (Symphonic Dialogue)

对白不只用来推动情节，更应揉合多个层次，如同交响乐一般由多个音轨共同鸣奏。

```
[第一轨道：故事对话／旋律]  ======> 处理当下的具体行动与情节
[第二轨道：道德对话／和谐]  ======> 处理背后的价值观对立与道德辩论
[第三轨道：关键字、潜文本／主导旋律] ===> 处理角色内心、幽灵与象征物
```

### 1. 第一轨道：故事对白／旋律 (Story Dialogue/Melody)
* **功能：** 推行情节，表达角色的欲望、计划与直接争执。这是对白的最表层，通常最为写实。

### 2. 第二轨道：道德对白／和谐 (Moral Dialogue/Harmony)
* **功能：** 探讨角色的价值取向和处世之道。当盟友攻击主角或主角与对手对峙时，角色会为自己的道德立场进行合理化辩护。这赋予对话思想的厚度。

### 3. 第三轨道：关键字词、特色短句及声音／主导旋律 (Keywords, Motifs, Subtext)
* **功能：** 重复出现的词汇或短语，承载着特殊的、随着情节发展而变化的情感意义。
* **实例：** 在《北非谍影》中，“巴黎（Paris）”、“通行证（Letters of Transit）”、“看着你，孩子（Here's looking at you, kid）”等关键语句，作为主导旋律（Leitmotif）在不同情境下反复出现，每次都带有不同的情感重音和潜文本。

### 4. 对话写作技巧
* **不要说教：** 即使是道德对话，角色也应站在自己的立场上进行“合理化辩护”，而非作者的说理。
* **对手的道德合理化：** 让对手的立场显得无比合理且有说服力，甚至能让主角哑口无言。这能极大增强戏剧张力（如《大审判》、《辣手摧花》）。
* **留白与静默：** 有时最震撼的对话是“不说出口的话”。
